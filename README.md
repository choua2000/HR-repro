# JavaScript Project

This is a basic JavaScript project with Node.js.

## Installation

Install dependencies using npm:
```bash
npm install
```

## Running

Start the project:
```bash
npm start
```

Or use watch mode:
```bash
npm run dev
```

- `index.js` - Main entry point
